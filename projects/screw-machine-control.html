Industrial Automation

Screw Machine Control Panel

PT Ciptapresisi Sarana Teknik

Industrial control panel development using Mitsubishi PLC, Mitsubishi Inverter, and Weintek HMI including programming, electrical wiring, testing, and commissioning.

Project Overview

This project involved the development of a complete industrial control panel system for screw machine operation, integrating PLC control, inverter drive systems, and HMI interface for machine monitoring and operation.

Scope of Work

  • PLC programming and configuration
  • Mitsubishi Inverter setup and parameter adjustment
  • Weintek HMI configuration
  • Electrical panel wiring and assembly
  • Testing and commissioning

Technology Used

Mitsubishi PLC Mitsubishi Inverter Weintek HMI
